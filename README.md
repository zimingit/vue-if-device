# vue-if-device

**vue-if-device** is a simple vue plugin to show/hide node on mobile/desktop devices.

### Install

```cmd
npm install --save vue-if-device
```

### Quick Start

```js
import device from 'vue-if-device'
Vue.use(device)
```

### Methods

- v-if-device.mobile
- v-if-device.desktop.
